Acerca del Proyecto

Este proyecto es una prueba técnica que demuestra la implementación de una aplicación Angular para gestionar productos y transacciones, consumiendo una API REST.

La aplicación permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) tanto para productos como para transacciones, y mantiene actualizado el inventario automáticamente.

Tecnologías utilizadas:
Angular TypeScript Bootstrap 5 Reactive Forms HttpClient Standalone Components
